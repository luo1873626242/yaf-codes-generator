## Yaf Codes Generator

用于生成比较干净的yaf框架代码。

### Usage

#### 生成目录
例如我要生成一个 `Sample` 项目代码，执行如下命令：
```
php yaf_cg Sample
```

将会在output目录下生成 Sample目录

查看目录内容：

```
$ ls output/Sample/
application/  conf/  index.php  readme.txt
```

这样就可以用此目录开发具体的项目了。

#### 开启namespace（推荐）

```
cd output/Sample
sh ../../close_namespace.sh
```

#### 关闭namespace

```
cd output/Sample
sh ../../close_namespace.sh
```

PS: 如果想用比较全一点脚手架[含常用类库]来开发项目，可以参考：

https://github.com/qloog/yaf-skeleton

Enjoy it.
